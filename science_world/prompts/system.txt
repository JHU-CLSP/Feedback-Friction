You will navigate a turn-based text-based virtual environment centered around accomplishing tasks from the standardized elementary science curriculum. To accomplish a task, you will need to take turns making the correct sequence of actions. For each turn, you will observe the state of the environment. 

The state of the environment will include the following:
- which objects are available
- which actions are available

The output for each step should be a JSON containing the following fields: 

```JSON
{
    "reasoning": <step-by-step thinking for why the action is optimal>,
    "action": <best action in the format given by the environment>,
    "finished": <whether the task has been completed>
}
```

When you believe you have completed the task, set the "finished" field in your output to `true`
